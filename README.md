# 🐍 Snake Water Gun Game

A simple Python game based on the classic Snake-Water-Gun rules.

## Features

- Random computer choice
- User input validation
- Win/Lose/Draw result
- Play multiple rounds

## Technologies Used

- Python 3

## How to Run

```bash
python main.py
```

## Rules

- Snake drinks Water 🐍 > 💧
- Water damages Gun 💧 > 🔫
- Gun kills Snake 🔫 > 🐍
